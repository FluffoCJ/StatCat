# StatCat - Work in progress system fetch written in rust.
If you wanna try this for whatever reason create a config file at ~/.config/statcat/config.toml and copy the example config.


#### Features
##### Implemented
- Kernel
- Distro Name
- Hostname
- Username
- Package count
- CPU_Brand Module
- Shell
- Gpu Module
- Uptime module
- Terminal module
- Icons
- Color
- Date time module

##### Work in Progress
- Memory Module

# Features
- Figlet
- Allow hex codes for colors
- Custom separator in config
- Colors module
- Distro Name
- Hostname
- Username
- Package count
- CPU Module
- Shell
- Gpu Module
- Uptime module
- Terminal module
- Icons
- Color
- Date time module
